# CSS

CSS stands for cascading style sheets and is used to help format html pages. It comes in three forms: external, internal, inline.

A simple example is:

```
h1 {
    color: red;
    font-size: 5pt;
}
```

### Common CSS Properties

 - color
 - align
 - font-size
 - font-family
